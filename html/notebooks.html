Notebook

El Notebook IPython ahora se conoce como el Cuaderno Jupyter. Es un entorno computacional interactiva, en la que se puede combinar la ejecución de código, texto enriquecido, las matemáticas, las parcelas y multimedia. Para más detalles sobre la Notebook Jupyter

El Notebook Jupyter

El Jupyter Notebook es una aplicación web que te permite crear y compartir documentos que contienen código en vivo, ecuaciones, las visualizaciones y texto explicativo. Los usos incluyen: limpieza y transformación de datos, simulación numérica, la modelización estadística, aprendizaje automático y mucho más.

Idioma de su elección

El ordenador dispone de soporte para más de 40 lenguajes de programación, incluidos los populares en Ciencias de datos como Python, R, Julia y Scala.

Cuadernos de Acciones

Cuadernos pueden ser compartidos con otros usuarios mediante correo electrónico, Dropbox, GitHub y el Notebook Jupyter Visor .

Widgets interactivos

El código puede producir una salida rica tales como imágenes, vídeos, látex, y JavaScript. widgets interactivos se pueden utilizar para manipular y visualizar los datos en tiempo real.

Integración de datos grandes

Aprovechar las herramientas de datos grandes, como Spark Apache, desde Python, R y Scala. Explora los mismos datos con los pandas, scikit-learn, ggplot2, dplyr, etc.
